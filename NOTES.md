# Notas

## Primer ejemplo
- Inicializar repo vacio
- Configurar correo y nombre
  - `git config --global user.name "Rafael Castillo"`
  - `git config --global user.email "rafael@trfs.me"`
- Escribir el hola mundo en c
- Compilarlo
- Hacer `git add .`
- Usar `git status`
- `git rm a.out`
- `git commit`
- Cambiar pendejadita
- Agregar un gitignore
- Commit, usar el `-m`
